## About

Implementation of group reactions - a shoal. Brains run away from the zombies but try to stay in the group.  
It uses the pygame library.

## How to install

* Download repository.
* Make sure that you have pygame library installed:  
  ``` 
  pip show pygame 
  ```  
  
  If not, install pygame:  
  ```
  pip install pygame 
  ```
* Run the program:  
  ```
  python Area.py
  ```
